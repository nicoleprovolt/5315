What is Docker

Docker creates 'containers' that you can use to build an application. In the container the devloper can ealiy house all the necessary parts of the applicaiton. The containers can be easily copied and run on different operating systems. https://opensource.com/resources/what-docker

What is Nginx

NGINX is open source software that is used as a webserver, reverse proxy, load balancer, and an email proxy server. NGINX was orginally used a webserver. Then, it was found that NGINX was able to handle a large number of connections, so developers started to use NGINX to balance loads and as a reverse proxy. NGINX has been developed to be one of the fastest and most powerful servers on the market. https://www.nginx.com/resources/glossary/nginx/

What is VM (virtual machine)

A virtual machine is basically a computer inside of another computer.It is an enitre computer system that runs inside of a program window, just like you would run any other applications on your computer. When the user is on the VM, the user interface looks like the same interface as a local computer. The VM creates a barrier inside of itself, so that none of the software or files on the VM can escape and infest the local machine. https://azure.microsoft.com/en-us/overview/what-is-a-virtual-machine/.

Who is Cloud Computing Service Provider - Digitalocean

DigitalOcean is a cloud compting company that hosts a IssS platform for developers. Through DigitalOcean developers create VM to build thier applications on. Developers have full control on how the VM is created. They can chose everything from the size, operating system, and even geographic area that thier data will be hosted. Once the VMs are created, the Digitalocean platform allows developers to easily manage and update thier VMs. https://searchcloudcomputing.techtarget.com/definition/DigitalOcean
